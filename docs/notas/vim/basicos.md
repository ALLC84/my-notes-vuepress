# Básicos

| COMANDO                    | ACCIÓN                           |
| -------------------------- | -------------------------------- |
| `vim`                      | Entrar                           |
| `edit path/nombre-archivo` | Editar archivo ya estando dentro |
| `vim path/nombre-archivo`  | Abrir archivo                    |
| `:help`                    | Ayuda                            |
| `:q`                       | Salir                            |
| `:q!`                      | Salir sin guardar                |
| `:w`                       | Guardar                          |
| `:wq`                      | Guardar y salir                  |
