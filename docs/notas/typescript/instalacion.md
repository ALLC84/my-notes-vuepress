# Instalación e Inicio

## Instalación

`npm install -g typescript`

## Inicia un proyecto

Creamos el archivo tsconfig.json
`tsc -init`

## Básico

### Compilar ts

`tsc <archivo.ts>`

### Modo observador de un archivo

`tsc --watch <archivo.ts>`

### Modo observador para todos los archivos .ts

`tsc *.ts -w`
