# Funciones

`repeat`

```css
grid-template-rows: repeat(2, 50%)
grid-template-columns: 50% repeat(2, 25%)
```

Shorthands

`grid: rows / columns`

```css
grid: repeat(2, 50%) / 50% repeat(2, 25% 10%);
```
