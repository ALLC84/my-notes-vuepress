# Integrar un proyecto de GitHub

### Integrando repositorios local con repositorio Git (facil)

## Vincular
```sh
git remote add origin https://github.com/...
```
## Mostrar vinculos
Ver vinculacion del proyecyo
```sh
git remote -v
```
## Elimina vinculos
Eliminar vinculacion
```sh
git remote remove origin
```
